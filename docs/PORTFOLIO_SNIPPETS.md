## LinkedIn Post

I completed an end-to-end Canadian retail credit risk analytics portfolio project.

The project goes beyond basic classification. It covers:

- Multi-sheet Excel ingestion and record-grain validation
- Data-quality assessment and missingness profiling
- Portfolio monitoring and borrower segment analysis
- Leakage-controlled feature engineering

This project was designed to reflect real finance analytics concerns: data quality, leakage prevention, explainability, operational capacity, and governance.
